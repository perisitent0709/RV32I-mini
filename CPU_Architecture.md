# RISC-V CPU架构图

## 整体架构
```
         指令存储器(IMEM)
                ↓ 32位指令
         指令解码器(decoder)
          ↓     ↓     ↓     ↓
      opcode  rs1/rs2  rd   imm
          ↓     ↓     ↓     ↓
       控制信号 ←寄存器堆→ ALU ← 立即数
                  ↓     ↓
              数据存储器(DMEM)
                  ↓
                写回寄存器
```

## 5个核心模块

### 1. 寄存器堆 (regfile.v)
- **功能**: 32个32位寄存器存储
- **接口**: 2读1写端口
- **特点**: x0寄存器硬编码为0

### 2. ALU (alu.v)
- **功能**: 算术逻辑运算
- **支持操作**: ADD, SUB, AND, OR, XOR, SLL, SRL, SRA
- **输出**: 32位结果 + 零标志

### 3. 指令存储器 (imem.v)
- **功能**: 存储程序指令
- **容量**: 1024条32位指令
- **访问**: 根据PC地址输出指令

### 4. 数据存储器 (dmem.v)
- **功能**: 数据读写存储
- **容量**: 1024字节
- **访问**: 支持字（32位）读写

### 5. 指令解码器 (decoder.v)
- **功能**: 32位指令解码
- **输出**: 寄存器地址、立即数、控制信号
- **支持**: R/I/S/B/J型指令格式

## 数据通路
1. **取指**: PC → IMEM → 指令
2. **解码**: 指令 → decoder → 控制信号+操作数
3. **执行**: 操作数 → ALU → 结果
4. **访存**: ALU地址 → DMEM → 数据（可选）
5. **写回**: 结果/数据 → 寄存器堆

## 支持的15条指令
- **算术(3)**: ADD, SUB, ADDI
- **逻辑(6)**: AND, OR, XOR, ANDI, ORI, XORI
- **移位(3)**: SLL, SRL, SRA
- **访存(2)**: LW, SW
- **分支(2)**: BEQ, BNE
- **跳转(2)**: JAL, JALR (可选)
